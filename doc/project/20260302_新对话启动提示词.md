## 💡 使用技巧

### 技巧 1：分步优化
不要一次性优化所有内容，分步进行：
```
第 1 步：优化 Phase 1（本次对话）
第 2 步：优化快速跳过机制（下次对话）
第 3 步：优化 Phase 5（再下次对话）
```

### 技巧 2：先理解再优化
让 Agent 先理解当前方案，再提出优化建议：
```
第 1 步：请阅读文档，总结当前方案的核心内容
第 2 步：请分析当前方案的优缺点
第 3 步：请提出优化建议
```

### 技巧 3：提供反馈
在 Agent 提出建议后，提供你的反馈：
```
你的建议很好，但是我有以下疑问：
1. [疑问 1]
2. [疑问 2]

请进一步说明。
```

### 技巧 4：参考历史
如果之前有类似的优化经验，可以参考：
```
之前我们优化过 [某个功能]，采用了 [某种方法]，效果很好。
这次能否采用类似的方法？
```
