## 📊 知识点总结

### 核心流程
- 10 个步骤：读取配置 → 筛选 → 读取模块配置 → 合并 → 计算激活 → 校验依赖 → 排序 → 加载 Steering → 整合 → 输出日志

### 配置管理
- 顶层配置优先级更高
- 模块配置作为补充
- 嵌套字典深度合并

### 依赖管理
- 依赖必须在激活列表中
- 自动检测循环依赖
- 依赖缺失会阻止加载

### 激活条件
- always: 始终激活
- directory_match: 目录匹配
- file_type_match: 文件类型匹配
- and: 所有条件满足
- or: 任一条件满足

### 排序规则
- 按 priority 从高到低
- 优先级相同按字母顺序
