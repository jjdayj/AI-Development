### 修改 1：脚本配置文件路径
**文件**：`.kiro/modules/ai-dev/v1.0/scripts/ai-dev-script.py`

**修改前**：
```python
config_path = os.path.join(os.path.dirname(__file__), "ai-dev-config.json")
```

**修改后**：
```python
# 支持模块化目录结构：从 ../configs/ 查找配置文件

config_path = os.path.join(script_dir, "..", "configs", "ai-dev-config.json")

```

**原因**：支持模块化目录结构，配置文件独立管理
