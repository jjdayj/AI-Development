## 🔍 属性验证详情

### 属性 5：优先级排序正确性

**形式化定义**：
*对于任意*激活模块列表，排序后的列表应该满足：
1. 对于列表中任意两个相邻模块 A 和 B（A 在 B 前面），A 的优先级 >= B 的优先级
2. 如果 A 和 B 的优先级相同，则 A 的名称字母顺序 <= B 的名称字母顺序

**验证方法**：
```python
# 规则 1：优先级递减验证
for i in range(len(sorted_modules) - 1):
    current_priority = sorted_modules[i]['priority']
    next_priority = sorted_modules[i + 1]['priority']
    assert current_priority >= next_priority

# 规则 2：相同优先级字母顺序验证
if current_priority == next_priority:
    current_name = sorted_modules[i]['name']
    next_name = sorted_modules[i + 1]['name']
    assert current_name <= next_name
```

**测试数据生成策略**：
- **模块名称**: 真实 Kiro 模块名 + 随机生成名称
- **版本号**: 标准 SemVer 格式（v1.0, v1.1, v2.0 等）
- **优先级**: 0-300 范围内的整数
- **唯一性**: 确保模块名称唯一，避免重复
