# 实现计划：Git 提交自动化联动

## 概述

本实现计划将 Git 提交自动化联动功能分解为一系列可执行的编码任务。实现采用 Python 脚本 + Prompt 的混合方式，核心逻辑使用 Python 实现，用户交互和流程控制使用 Prompt。
