## 命名原则

### ✅ 推荐做法
1. **简短易记**：越短越好，方便他人记忆和输入
2. **专业性**：适合工作和正式场合使用
3. **唯一性**：容易区分，不易混淆
4. **可读性**：使用点号分隔提高可读性
5. **国际化**：使用拼音便于国际交流

### ❌ 避免做法
1. 避免使用下划线（`_`）：不够美观
2. 避免过多数字：显得不专业
3. 避免复杂符号：只用字母、数字、点号
4. 避免随意昵称：如果用于工作场合
5. 避免过长：超过15个字符不易记忆
