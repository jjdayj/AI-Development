## GitHub 注册步骤

### 1. 访问 GitHub 注册页面
打开浏览器，访问：https://github.com/signup

### 2. 输入邮箱
- 输入：yangzhuo.dev@gmail.com
- 点击"Continue"

### 3. 创建密码
- 设置强密码（至少8个字符，包含数字和字母）
- 建议使用与 Gmail 不同的密码
- 点击"Continue"

### 4. 输入用户名
推荐用户名：
- `yangzhuo` - 简洁直接
- `yang-zhuo` - 带连字符
- `yangzhuo-dev` - 带开发者标识
- `zhuoyang` - 姓名倒置

注意：用户名将成为你的 GitHub 主页地址：https://github.com/[用户名]

### 5. 邮箱验证
- GitHub 会发送验证码到你的 Gmail
- 打开 Gmail App 查看邮件
- 输入验证码

### 6. 完成人机验证
- 可能需要完成"我不是机器人"验证
- 按提示操作

### 7. 选择计划
- 选择"Free"（免费版）
- 功能已经很强大，适合个人使用

### 8. 完成注册
- 可能会有一些个性化问题（可以跳过）
- 注册成功后会看到 GitHub 欢迎页面
