# 谷歌账号注册指南

## 操作时间
2026年3月1日 14:46:00

## 前置条件
- 有效的手机号码（用于接收验证码）
- 稳定的网络连接
- 浏览器（推荐 Chrome、Firefox、Edge）

## 注册步骤

### 1. 访问谷歌账号注册页面
打开浏览器，访问：https://accounts.google.com/signup

### 2. 填写基本信息
- **名字**：输入你的名字
- **姓氏**：输入你的姓氏
- **用户名**：选择一个唯一的用户名（这将成为你的 Gmail 地址：用户名@gmail.com）
- **密码**：设置强密码（至少8个字符，包含字母、数字和符号）
- **确认密码**：再次输入密码

### 3. 手机验证
- 选择国家/地区代码（中国：+86）
- 输入手机号码
- 点击"下一步"
- 输入收到的验证码

### 4. 完善账号信息
- **辅助邮箱**（可选）：用于账号恢复
- **出生日期**：输入你的出生日期
- **性别**：选择性别

### 5. 隐私条款
- 阅读 Google 服务条款和隐私政策
- 点击"我同意"

### 6. 完成注册
注册成功后，你将获得：
- Gmail 邮箱地址：[你的用户名]@gmail.com
- Google 账号访问权限

## 注意事项
1. 用户名一旦设置无法更改，请谨慎选择
2. 密码建议使用密码管理器保存
3. 建议开启两步验证以提高安全性
4. 记录下你的 Gmail 地址，后续注册 GitHub 时需要使用

## 下一步
完成注册后，请确认你的 Gmail 地址，然后我们将进行第二步：使用该邮箱注册 GitHub 账号。
